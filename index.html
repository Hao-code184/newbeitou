FAQ 常見問題

  • Q : 新北投有什麼必訪的景點？

    A: 地熱谷、北投溫泉博物館與北投公園都是不可錯過的景點，您還可以探索當地特色溫泉飯店與步道。

  • Q : 地熱谷的參觀時間與費用是多少？

    A: 地熱谷開放時間為上午9點至下午5點，免費參觀，但請注意每週一休館。

  • Q : 北投有提供哪些溫泉體驗？

    A: 新北投提供公共湯屋、溫泉飯店以及露天溫泉等多樣選擇，適合個人或家人一同享受。

  • Q : 如何前往新北投？

    A: 可搭乘台北捷運淡水信義線至北投站，再轉乘新北投支線，約5分鐘即可抵達新北投站。(地圖 : 點我前往 )
